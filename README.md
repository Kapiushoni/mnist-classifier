# Проект "Класифікатор рукописних цифр MNIST"

Це мій перший end-to-end проект у сфері Computer Vision, реалізований як частина інтенсивної самопідготовки.

**Мета:** Побудувати, натренувати та оцінити модель машинного навчання для розпізнавання рукописних цифр з набору даних MNIST.

### Технічний стек

* **Python**
* **Scikit-learn:** Для завантаження даних, поділу вибірки та побудови моделі.
* **Matplotlib:** Для візуалізації даних та результатів.
* **Jupyter Notebook:** Як середовище для прототипування.

### Етапи проекту

1.  **Завантаження та Підготовка:** Дані (70,000 зображень) були завантажені з OpenML та нормалізовані (шкалування пікселів 0-255 -> 0-1).
2.  **Візуалізація:** Проаналізовано приклади цифр з набору даних.
3.  **Побудова моделі:** Використано **Логістичну Регресію** (`LogisticRegression` зі `sklearn`) як швидку та ефективну базову модель.
4.  **Оцінка:** Модель досягла **~90% точності** на тестовій вибірці (10,000 зображень).
5.  **Аналіз:** Побудовано матрицю помилок (Confusion Matrix) та візуалізовано конкретні приклади, де модель помилилася. Аналіз показав, що модель переважно плутає візуально схожі цифри (напр., '4' і '9').

### Результати

У файлі `mnist_classifier.ipynb` міститься повний код, що включає:
* Завантаження та візуалізацію даних.
* Тренування моделі.
* Розрахунок точності.
* Візуальний аналіз помилок (де модель сплутала цифри).
